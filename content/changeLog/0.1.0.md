# 更新日志

> 本网站所有功能更新、优化与修复记录。

## 🎉 v0.1.0 - 2025-11-26

### ✨ 新特性

- **新增切换字体功能**：现在可以在设置中自由切换“阿里妈妈方圆体”与“数黑体”，打造个性化阅读体验。

  ![演示图片](https://www.xiaohev.com/files/74183bc2c016e4083ae8a94cb65e3882.webp)

### ⚡️ 优化与修复

- **体验优化**：新增了字体下载时的 **Loading 状态提示**，让加载过程不再枯燥。
- **性能优化**：重构了字体加载逻辑，利用 `FontFace` API 实现**按需下载**，大幅节省带宽。
- **修复**：修复了在线上环境下点击字体可能无反馈的问题。
