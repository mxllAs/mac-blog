# 更新日志

> 持续优化系统体验，增强内容连接能力。

## 🎉 v0.1.1 - 2025-11-27

本次更新重点带来了 macOS 标志性的聚焦搜索体验，并开放了 RSS 订阅源，让获取信息变得前所未有的简单。

### ✨ 核心特性

- **🔍 Spotlight 聚焦搜索**：
  - 按下 `Cmd + K` (Mac) 或 `Ctrl + K` (Win) 即可随时呼出全局搜索框。
  - 支持搜索 **本地应用**（如关于、日志）和 **在线文章**。
  - 采用全屏毛玻璃遮罩设计，沉浸感十足。
  ![演示图片](https://www.xiaohev.com/files/f9efc694a49fdb6638ad5289eebcfec0.webp)
- **📡 RSS 订阅支持**：
  - 顶部栏新增 RSS 订阅入口，点击即可获取标准 XML 订阅源。
  - 支持主流 RSS 阅读器（如 Feedly, Reeder）抓取，不再错过任何更新。

### ⚡️ 优化与修复

- **体验优化**：优化了搜索框的快捷键逻辑，解决了 SSR 环境下的兼容性问题。
- **SEO 增强**：完善了 Sitemap 网站地图生成逻辑，支持动态文章分页抓取。
- **交互细节**：为顶部栏图标添加了 Hover 提示与颜色反馈，同时更新顶部图标。
