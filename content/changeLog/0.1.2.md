> 让 Web 博客拥有操作系统的生命力。

## 🎉 v0.1.2 - 2025-11-28

本次更新带来了完整的 **拟物化系统更新** 流程，完善了 Web OS 的沉浸式交互拼图。

### ✨ 核心特性

- **⚙️ 拟物化软件更新**：
  - 在“设置”中新增了 **软件更新 (Software Update)** 模块。
  - 真实模拟了 macOS 的检查更新、下载进度条与系统重启（刷新）动画，仪式感拉满。
  - 智能版本管理：系统会对比本地“安装”的版本与线上最新版，只有点击“立即升级”才会应用更新。

- **🚨 智能状态通知**：
  - Dock 栏设置图标新增醒目的红色 **NEW** 徽标，第一时间提示新版本。
  - 优化了红点消除逻辑：查看日志不再消除红点，只有真正执行升级操作后才会标记为已读，逻辑更加严谨。

### ⚡️ 优化与修复

- **逻辑增强**：重构了 `app.vue` 的全局版本检测机制，解决了版本号格式不统一导致红点无法消除的问题。
- **界面优化**：优化了设置菜单的视觉风格，使用了更精致的徽标样式替代原有的圆点。
